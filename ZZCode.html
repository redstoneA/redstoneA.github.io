ZZ Code (V1.0) 简介

ZZ Code (V1.0) 是一个基于 Tkinter 的简单代码编辑器，旨在为用户提供一个轻量级且易于使用的编程环境（真的“轻量级而且易于使用”，使我的代码有问题都不知道在哪）。该编辑器支持基本的文件操作（如打开、保存和运行代码），特别适用于 Python 脚本的编写和执行。

主要功能

文本编辑

提供了一个文本编辑区域，用户可以在其中编写或修改代码。

文件操作

打开文件：用户可以选择一个文件并将其内容加载到编辑器中。

保存文件：用户可以将当前编辑的内容保存到指定的文件中，支持自动添加 .py 扩展名。

运行代码：用户可以运行当前编辑的 Python 脚本，并查看运行结果。

注意事项

1. 该编辑器仅支持 Python 脚本的编写和执行，不支持其他编程语言。

2. 由于开发者懒，ZZCode是一个py文件，需要用python运行。

3. 当前版本的ZZCode不支持第三方模块（本站开发者：ZZCode官方你做了个什么？）。
